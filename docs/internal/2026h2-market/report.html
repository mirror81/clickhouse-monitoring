Bottom line: the wedge is real and unoccupied — nobody ships a pganalyze-style scored advisor for ClickHouse. The price is right; what's broken is collection (AI overage never bills, 402s return raw JSON, no billing UI). Turning collection on is found money and the highest-ROI work in the roadmap.

1 · The five findings

  1. Clean, unoccupied wedge. Competitors are raw dashboards (Grafana, ch-ui, clickhouse-monitoring), human consulting (Altinity), reactive LLM chat (ClickHouse Cloud Ask AI), or preview-grade telemetry (Datadog DBM is preview-only for ClickHouse as of July 2026). None gives continuous, scored, prescriptive tuning.
  2. The pains are tuning problems, not bugs — exactly what an advisor monetizes: too-many-parts, MEMORY_LIMIT_EXCEEDED, merge storms, expensive mutations, roll-your-own alerting, Cloud cost anxiety.
  3. Pricing is well-anchored. $29/$99 undercuts pganalyze ($149/$399) ~5× and Datadog DBM (~$70/host). Approve the anchors; fix the leaks.
  4. The MCP server is a distribution channel competitors lack — list on MCP Registry, PulseMCP, cursor.directory, Smithery.
  5. Watch ClickHouse Cloud's native AI — the one real threat. Moat: the self-hosted / Altinity / BYOC surface + fleet view they structurally won't serve.

3 · Billing verdict & optimizations

Fix collection first

  • P0 AI overage never accumulates → bills $0 past allowance
  • P0 402 returns raw JSON, not a paywall — biggest CVR leak
  • P0 No in-app billing surface (plan/usage/renewal)
  • P1 Per-host overage unplugged — hosts hard-cap

Infra already exists → highest ROI in the roadmap.

Then optimize

  • Explicit included-host counts (Pro 1 / Max 3) — kill sticker shock
  • Replica = 0.5 host (copy pganalyze)
  • Lead with BYOK on the AI advisor (2026 expectation)
  • Test a $199 Fleet mid-anchor for 5–10 host clusters
  • Annual (~2 months free) after e2e tests pass
Will people pay? Yes — conditionally on (1) an advisor recommendation good enough to trust on first run, (2) a visible purchase path, (3) a free tier with real standalone value. The paying segment exists and pays more than we ask.
